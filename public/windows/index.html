CHECKLIST · Part 05

签约、安装、验收
完整 Checklist

把签约前、合同内容、安装当天、验收后续整理成一张可执行清单。可以打印或下载后，按阶段逐项核对。

阶段 Phase 检查项目 必须 Must
🔍 签合同之前 查地址是否在遗产区（montreal.ca），确认允许的材料和颜色 必须
🔍 签合同之前 如需要，申请市政施工许可证，等批准后才动工。同尺寸更换（Replace doors and windows with models that have the same dimensions）也不等于自动免 permit，尤其是遗产区、特殊建筑、材料/颜色/样式变化时。 必须
🔍 签合同之前 核实 总公司的RBQ 和安装商的RBQ（rbq.gouv.qc.ca），确认有效且覆盖门窗安装。去 RBQ 官网查询该公司是否有未处理的索赔 必须
🔍 签合同之前 确认责任险 ≥ $200万，要求提供保险证明 必须
🔍 签合同之前 如申请 Rénoclimat 补贴，预约初次评估，动工前完成 必须
🔍 签合同之前 同时向至少 3–5 家公司发询价单，不要只问一家 可选
🔍 签合同之前 确认窗户类型和开启方式——主动问所有可选类型，不完全依赖销售建议 可选
🔍 签合同之前 确认产品通过 ENERGY STAR Zone 3 和 CSA A440 认证 可选
📋 合同内容检查 明确写明完工日期，以及延误超过 X 天后的违约赔偿或退款权利 必须
📋 合同内容检查 确认付款结构：定金(10-30%) + 交付前/当天(60%) + 验收尾款(10%). 在验收合格后支付. 可以考虑总款项的10%作为尾款，直到安装100%完成、杂物清运完毕、且业主签署最终验收单确认无缺陷或损坏后支付。 必须
📋 合同内容检查 写明谁来安装：自有团队还是外包？外包商的 RBQ 执照号是什么？ 必须
📋 合同内容检查 产品保修含零件+人工，安装保修，密封玻璃保修——分别写明年限和责任方 必须
📋 合同内容检查 含变更单条款：施工中任何额外费用必须提前书面确认，不得当场强收 必须
📋 合同内容检查 品牌公司对外包安装质量负连带责任（防止安装商倒闭后无人负责） 必须
📋 合同内容检查 逐行核对材料品牌、型号、尺寸、颜色，与确认的完全一致 可选
📋 合同内容检查 施工范围含：旧窗拆除清运、外部打胶、室内收边、现场清洁 可选
🔧 安装当天 安装前拍照记录全屋墙体、地板、窗台状况（保留时间戳证据） 必须
🔧 安装当天 如当天突然要求额外收费，要求对方出具书面变更单，不要口头同意 必须
🔧 安装当天 确认来的是合同里说的安装团队，或核实临时换人的团队 RBQ 执照 可选
🔧 安装当天 保留窗户上的 ENERGY STAR 标签（申请 Rénoclimat 补贴时需要） 可选
🔧 安装当天 安装完成后，当场逐一测试每扇窗门的开关、锁定功能，不合格考虑拒绝签收 可选
🔧 安装当天 ① 水平和垂直度
用水平仪检查每扇窗框是否水平、窗侧框是否垂直。窗框歪了会导致日后越用越难开，密封条也会受力不均匀提前老化。特别注意转角位置和大尺寸窗户——这两类最容易歪。若有缺陷，自动触发“扣留尾款”机制		 必须
🔧 安装当天 ② 开关功能逐一测试
每扇窗、每扇门都要亲手开关三次以上。检查：

开关是否顺畅，有没有卡涩或需要用力
关闭后有没有明显缝隙（拿一张纸插边缘测试，能轻松抽出说明密封不严）
锁定后锁扣是否咬合紧实，不松动

门要特别测试从里外两侧都能正常开关。若有缺陷，自动触发“扣留尾款”机制		 必须
🔧 安装当天 ③ 金属件完整性
逐一检查对照合同上的规格：

把手、锁芯、铰链、纱窗扣件是否齐全
把手转动是否顺滑，锁芯插拔是否正常
铰链有没有松动或安装歪斜
纱窗是否能正常拆装，尺寸是否合适（纱窗太小一有风就会被吹开）
金属件质量在现场很难深度检测，但可以看：表面有没有毛刺、涂层有没有刮花、转动部件有没有异响。若有缺陷，自动触发“扣留尾款”机制		 必须
🔧 安装当天 ④ 玻璃检查
在自然光下从不同角度看每块玻璃：

表面有没有划痕或裂纹
有没有气泡或明显的变形（透过玻璃看直线是否仍是直线）
双层/三层玻璃内侧有没有起雾或水汽（说明密封层已失效）
玻璃边缘与窗框之间的密封胶是否连续、饱满，有没有断点。若有缺陷，自动触发“扣留尾款”机制		 必须
🔧 安装当天 ⑤ 密封和隔热
用手沿窗框四周慢慢移动，感受有没有冷气渗入。如果是冬天施工，可以用打火机或蜡烛靠近边缘，火焰飘动说明漏风。外部的硅胶打胶要连续无断点，室内的泡沫填充要饱满。若有缺陷，自动触发“扣留尾款”机制		 必须
🔧 安装当天 ⑥ 排水坡度
检查窗台或门槛是否向外有轻微倾斜（约5°），确保雨水往室外排走，不会积水或倒灌进屋。若发现水平或向内倾斜，自动触发“扣留尾款”机制。		 必须
🔧 安装当天 ⑦ 防水膜 / flashing
确认底部 sill 以及两侧 jamb 下部是否安装防水膜或自粘防水层，尤其是外墙开口位置。只靠打胶不能替代防水膜，长期漏水往往从这里开始。若有缺陷，自动触发“扣留尾款”机制。		 必须
🔧 安装当天 ⑧ 保温材料使用方式
泡沫填充应连续、均匀，但不能过度膨胀到挤压窗框；如使用矿棉，不能被压得太实，否则会失去保温效果，也可能让窗框受力变形。若有缺陷，自动触发“扣留尾款”机制。		 必须
🔧 安装当天 ⑨ 预留缝隙 / expansion gap
窗框与墙体开口之间应保留合理安装间隙（约3mm / 1⁄8 inch，视产品要求而定），不能硬塞或过紧。过紧会影响热胀冷缩，日后可能导致卡顿、开裂或密封条提前老化。若有缺陷，自动触发“扣留尾款”机制。		 必须
🔧 安装当天 ⑩ 与安装人员共同验收
验收必须在安装人员离开前完成。发现问题当场拍照、编号、写进验收备注，并要求对方确认整改方式和时间；不要等团队离开后才第一次检查。		 可选
🔧 安装当天 ⑪ 外观和施工现场

窗框和墙面之间的收边是否整齐，没有过大的缝隙
室内外的收边条颜色是否和合同约定的一致
玻璃和框上有没有残留的胶纸痕迹、打胶残余（有些安装工会留下，后期很难清除）
现场是否清洁——旧窗碎片、包装材料、工具是否已清走。
🔧 安装当天 发现问题怎么处理
当天拍照，当天发邮件给承包商，列清楚问题编号和照片。不要当天签收验收单，等问题解决后再签。
如果承包商要求您当天付尾款，可以说：「我需要确认所有问题已记录并得到书面回复后再付尾款，这是合同里约定的验收条款。」		 可选
✅ 验收与后续 拍照记录所有窗户安装完成状态（带日期戳） 必须
✅ 验收与后续 维修后结算：直到所有维修完成，再释放最后的 10%
✅ 验收与后续 发现任何问题，当天发书面邮件通知（不要只打电话），建立证据链 必须
✅ 验收与后续 收到发票后核对：金额、型号、数量是否与合同一致 必须
✅ 验收与后续 预约 Rénoclimat 二次评估（Post-Retrofit） 可选
✅ 验收与后续 索取产品质保书和安装质保书原件，妥善保存 可选
✅ 验收与后续 一年后做一次检查：缝隙均匀度、密封条完好度、有无起雾或结冰 可选
⚖️ 出了问题怎么办 所有沟通发送书面邮件，保留时间戳——如果邮件无效，下一步是发送挂号信形式的“正式催告函” 必须
⚖️ 出了问题怎么办 向 RBQ 投诉承包商（rbq.gouv.qc.ca）——RBQ可暂停/吊销执照，施压最强 可选
⚖️ 出了问题怎么办 向 OPC 投诉（Office de la protection du consommateur） 可选
